For using the seaters integration in production you should consider the following points:

  • Load the script tag, referring to seaters.bundle.js, only once in the document head
  • The css and button assets can be copied locally for better performance
  • Customize the join button even further by placing the on-click="SeatersSDK.joinWl(...)" on any element you want
